Python Version: Python 2.7.17

To run the program:
First, run Bob.py
For example, to run Bob on port 2100:
	python Bob.py 2100 -v

Next, run Alice.py
For example, to run Alice to connect to port 2100 and localhost, with message of "hello":	
	python Alice.py localhost 2100 hello -v

-h for help
	For Example: python Alice.py -h
-v for verbose detailed output
